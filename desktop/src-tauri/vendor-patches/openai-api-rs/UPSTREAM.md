## Update Workflow

1. Sync vendor from upstream reference and re-apply local patches:
   - `./scripts/vendor-openai-api-rs-sync.sh --verify`
2. If local customizations changed, export/update patch file(s):
   - `./scripts/vendor-openai-api-rs-export-patch.sh --output desktop/src-tauri/vendor-patches/openai-api-rs/0001-local.patch`
3. Update metadata lines above (`upstream_ref`, `upstream_tag`, `crate_version`, `last_synced_utc`).
4. Commit patch files + metadata.

- `upstream_ref` should be a concrete commit SHA.
- Patch application uses:
  - `patch -p0 -d desktop/src-tauri/vendor/openai-api-rs < <patch-file>`
